# Kilis Eczane Stok Sorgulama Sistemi

Bu proje, simülasyon verileri kullanılarak geliştirilmiş
bir masaüstü eczane stok sorgulama uygulamasıdır.

Kullanıcılar ilaç adına göre arama yaparak,
eczanelerdeki stok durumlarını ve nöbetçi bilgilerini
görüntüleyebilir.

## Özellikler
- İlaç adına göre arama
- Eczane bazlı tekil listeleme
- Stok durumu görüntüleme
- Nöbetçi eczane bilgisi
- Akademik ve sade arayüz tasarımı

## Kullanılan Teknolojiler
- Python
- Pandas
- CustomTkinter

## Veri Hakkında
Bu projede kullanılan veriler gerçeği yansıtmaz.
Simülasyon amaçlı üretilmiştir.
